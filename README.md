## 🔄 Development Workflow

1. **Create branch:** `git checkout -b feature/xxx`
2. **Develop:** Code trong branch
3. **Push:** `git push origin feature/xxx`
4. **PR:** Mở Pull Request
5. **Review:** Claude auto-review + Bố approve
6. **Merge:** Merge vào main

## 🤖 Claude Review

- Tự động review khi PR được tạo
- Comment `@claude` để yêu cầu re-review
- Review focus: code quality, security, architecture

## 📋 Review Checklist

- [ ] Code follows project conventions
- [ ] Tests included (if applicable)
- [ ] Documentation updated
- [ ] Claude review approved
- [ ] Bố approved
